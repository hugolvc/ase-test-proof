# Project Naissance: Enterprise Expense Approval System

**Objective:** Provide a comprehensive system for employees to submit expenses, managers to approve them, and finance teams to audit and process payouts.

**Stakeholders:** Employees, Managers, Finance Department.

**Scope:** A full-stack web application built with Next.js (App Router), leveraging Prisma ORM with SQLite for local data persistence, and Tailwind CSS for styling. The application must support customized theming out of the box with 10 exact predefined skins and specific functional color palette.
