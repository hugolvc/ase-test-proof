# Non-Functional Requirements Register

| ID | Requirement Description | Category |
|---|---|---|
| NFR-001 | Application must be contained within a single repository (Frontend, Backend, Database). | Architecture |
| NFR-002 | Application must be executable by a human reviewer using a single `npm run dev` command. | Usability / DevEx |
| NFR-003 | Application must use SQLite for local data persistence via Prisma ORM. | Persistence |
| NFR-004 | Styling must be implemented entirely using Tailwind CSS, supporting 10 distinct predefined skin themes. | Styling |
